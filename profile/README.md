# Welcome to wolf-infra

We build open-source infrastructure, routing, and monitoring tools for homelabs and edge environments.

## 🚀 Flagship Projects

* **[traefik-mesh-companion](https://github.com/wolf-infra/traefik-mesh-companion)**: Automates Cloudflare DNS and NetBird routing directly from Traefik Docker labels.
* **[gatus-api-bridge](https://github.com/wolf-infra/gatus-api-bridge)**: A lightweight HTTP API bridge for dynamically managing Gatus endpoints.

## 🛠️ Architecture Philosophy
Our tools are built with a focus on:
* **Statelessness**: Using files and labels as the source of truth.
* **Security**: Zero-trust networking (NetBird) and hardened Go binaries.
* **Simplicity**: Do one thing, and do it incredibly well.
