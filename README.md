# wolf-infra

This repository contains the centralized configurations and templates for the **wolf-infra** organization. 

By keeping these files centralized, we ensure consistency across all our microservices and infrastructure tools.

## Contents
* **`profile/`**: Contains the public-facing `README.md` for the organization page.
* **`workflows/`**: Reusable GitHub Actions for CI/CD, Docker building, and linting.
* **`ISSUE_TEMPLATE/`**: Standardized templates for bug reports and feature requests across all repositories.
